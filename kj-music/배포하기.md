# 📱 핸드폰으로 접속할 인터넷 주소 받기

아래 방법 중 **하나만** 따라 하시면 됩니다. 모두 무료입니다.

---

## 방법 1: Netlify 드래그 앤 드롭 (가장 쉬움)

1. **https://app.netlify.com** 접속
2. 회원가입 (이메일 또는 Google로 간단 가입)
3. 화면에 **"Drag and drop your site output folder here"** (폴더를 여기에 끌어다 놓으세요) 영역이 보임
4. 컴퓨터에서 **`kj-music` 폴더 전체**를 그 영역에 끌어다 놓기  
   (폴더 안에 `index.html`, `audio` 폴더가 있어야 함)
5. 잠시 후 **"Your site is live"** 와 함께 주소가 나옴  
   예: `https://random-name-12345.netlify.app`
6. 이 주소를 **핸드폰 브라우저에 입력**하면 바로 접속 가능

---

## 방법 2: GitHub Pages

1. **https://github.com** 접속 후 로그인(또는 회원가입)
2. 오른쪽 상단 **+** → **New repository** 클릭
3. **Repository name**에 `kj-music` 입력 → **Create repository** 클릭
4. 생성된 페이지에서 **"uploading an existing file"** 링크 클릭
5. **`kj-music` 폴더 안의 모든 파일**을 끌어다 놓기  
   (`index.html`, `audio` 폴더, README.md 등 모두)
6. 맨 아래 **Commit changes** 클릭
7. 상단 메뉴 **Settings** → 왼쪽에서 **Pages** 클릭
8. **Source**에서 **Deploy from a branch** 선택  
   **Branch**를 `main`(또는 `master`), 폴더를 **/ (root)** 로 선택 → **Save**
9. 1~2분 후 **"Your site is live at https://사용자이름.github.io/kj-music/"** 라고 나옴
10. 이 주소를 **핸드폰에서 입력**하면 접속 가능

---

## 방법 3: Vercel

1. **https://vercel.com** 접속 후 로그인(또는 GitHub로 가입)
2. **Add New...** → **Project** 클릭
3. **Import Git Repository** 대신, 화면에서 **"Upload"** 또는 **Vercel CLI** 사용 가능
4. (CLI 사용 시) 터미널에서:
   ```bash
   cd "c:\Users\park jiyeon\Desktop\Cursor\kj-music"
   npx vercel
   ```
   처음에는 로그인 안내가 나오면 브라우저에서 로그인 후, 다시 위 명령 실행하면 배포되고 주소가 나옵니다.

---

## 추천

- **컴퓨터를 잘 안 쓰시면** → **방법 1 (Netlify 드래그 앤 드롭)**  
- **GitHub 계정이 이미 있으면** → **방법 2 (GitHub Pages)**

배포 후 나온 주소(예: `https://~~~~.netlify.app` 또는 `https://사용자이름.github.io/kj-music/`)를  
**핸드폰 주소창에 그대로 입력**하면 같은 페이지가 열립니다.
