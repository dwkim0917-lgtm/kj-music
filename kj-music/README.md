# 아람북스 노래 웹사이트

노래를 듣는 간단한 웹 플레이어입니다.

## 기능
- **연속 재생**: 리스트 순서대로 끝까지 재생 후 처음부터 다시 재생
- **한 곡 반복**: 선택한 곡만 반복 재생
- 모바일에서 주소만 입력하면 바로 재생 가능

## 사용 방법

### 1. 노래 파일 넣기
`audio` 폴더에 MP3 파일을 넣어주세요.
- 01.mp3, 02.mp3, 03.mp3, 04.mp3, 05.mp3

곡 제목을 바꾸려면 `index.html`을 열어서 `tracks` 배열 안의 `title`을 수정하면 됩니다.

### 2. 컴퓨터에서 실행
`index.html`을 더블클릭해서 브라우저로 열면 됩니다.

### 3. 휴대폰에서 접속하기
같은 Wi‑Fi에 컴퓨터와 휴대폰을 연결한 뒤:

1. 터미널(또는 명령 프롬프트)을 열고 이 폴더로 이동:
   ```
   cd "c:\Users\park jiyeon\Desktop\Cursor\kj-music"
   ```
2. 간이 웹 서버 실행:
   ```
   npx -y serve -l 3000
   ```
   (Node.js가 없다면: `python -m http.server 3000`)
3. 컴퓨터에서 터미널에 나온 주소 확인 (예: http://192.168.0.10:3000)
4. 휴대폰 브라우저에서 **같은 주소**로 접속

**인터넷 주소로 핸드폰 접속:** → **`배포하기.md`** 파일을 열어서 따라 하시면 됩니다 (Netlify 드래그 앤 드롭이 가장 쉽습니다).
